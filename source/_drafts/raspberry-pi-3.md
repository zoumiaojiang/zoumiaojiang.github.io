## 前期准备

我手边没有 Windows 电脑，所以只能讲讲如何使用 Mac 电脑为树莓派3 安装操作系统了。

- 一个树莓派 3 的板子
- 一台 Mac 电脑
- 一个 SD 卡
- 一个读卡器
- 一个 HDMI 线 （非必须）
- 一个显示器（非必须）
- 一个 USB 3.0 线（非必须）
