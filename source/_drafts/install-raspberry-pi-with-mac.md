## 什么是树莓派

手拿一个树莓派板子，然后怎么开始玩起来呢？首先就是需要给板子安装一个操作系统，

## 前期准备

- 一个树莓派 3 的板子
- 一台 Mac 电脑
- 一个 SD 卡
- 一个读卡器
- 一个 HDMI 线 （非必须）
- 一个显示器（非必须）
- 一个 USB 3.0 线（非必须）
